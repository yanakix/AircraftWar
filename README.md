# AircraftWar 飞机大战

> Java 课程项目｜2D 飞行射击游戏｜面向对象设计与工程实践

## 项目简介

本项目是一个基于 **Java** 实现的 2D 飞行射击游戏，完成于校内课程 **《软件构造》**。项目实现了完整的游戏流程，包括菜单系统、难度分级、战斗逻辑与排行榜结算，注重代码结构设计与可扩展性，而不仅是功能堆叠。

## 功能特性

* 完整的游戏流程：开始菜单 → 游戏进行 → 结算与排行榜
* 多难度模式：不同敌机生成策略与数值配置
* 本地排行榜系统：游戏结束后进行得分统计与排名
* 音效系统：基础背景音与战斗音效支持

## 技术栈

* **语言**：Java
* **UI**：Swing（菜单与得分榜界面）
* **并发**：Java 线程池（定时任务调度、音频线程管理）
* **工具**：Git

## 设计与实现要点

### 架构设计

* 采用模块化设计，将 **游戏逻辑、UI、数据管理** 进行解耦
* 核心游戏流程通过模板方法统一管理，便于扩展不同难度模式

### 设计模式实践

* **单例模式**：英雄机对象采用饿汉式单例，保证全局唯一性与线程安全
* **抽象工厂模式**：统一创建不同类型的敌机、子弹与道具，提升扩展性
* **策略模式**：将射击行为抽象为策略，支持道具切换攻击方式
* **观察者模式**：用于实现炸弹道具对不同敌机产生不同效果
* **DAO 模式**：管理游戏得分数据，实现排行榜的持久化与解耦

### 并发与性能

* 使用线程池调度敌机生成、子弹发射等周期性任务，避免阻塞主线程
* 音频播放通过独立线程池管理，减少对游戏逻辑的影响
